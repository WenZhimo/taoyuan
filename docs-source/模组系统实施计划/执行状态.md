## 当前阶段

- 阶段 0：已完成。交付物包括导出符号级机器盘点、官方内容快照、哈希黄金向量、旧存档兼容夹具、最小合法/非法包夹具和可重复基线验证。
- 阶段 1：已完成。交付物包括 TypeBox 公开契约、Ajv 结构校验、语义校验、通用注册表、冻结生命周期、旧静态数据适配层、公开 JSON Schema 生成与一致性检查、快照 codec 和 10 万条注册/查找响应性验证。
- 阶段 2：已完成。已完成“物品、作物和配方试点”的端到端官方内容迁移闭环，范围限定为官方 Tag/Item/Crop/Recipe 查询门面、Crop 注册表适配、Recipe 公开契约等价、Cooking 材料规划读取链路、烹饪弹窗标签材料手动选料、组成标签传播和旧查询入口兼容；尚未进入第三方包加载器、模组安装界面、安装/卸载事务、存档环境绑定或机制插件。
- 阶段 3：已完成当前商店迁移闭环并通过阶段 3 总验收。已完成三个可回滚切片：官方 `taoyuan:shop` 商店定义契约、旧 `SHOPS` 适配器、只读查询门面、规范化快照和等价测试；官方 `taoyuan:shop_offer` 商品池显示字段、六大商店商品池投影、按季节查询和分组查询等价测试；以及 `ShopView.vue` 商品来源切换到注册表分组渲染。复杂购买条件仍由现有购买 handler 校验，后续在开放第三方商店商品前再抽象为声明式条件。
- Stage 4: in progress. First rollback slice extends official `taoyuan:enchantment` definitions with `rarity`, `randomWeight` and `effects`, adds read-only enchantment/drop-table facades, and proves legacy enchantment values plus monster drop-table projections remain equivalent. Subsequent slices add a shared drop-table rolling adapter and named equipment drop tables while keeping legacy monster item drops and equipment settlement behavior unchanged.
