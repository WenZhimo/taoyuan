# 桃源乡存档编辑器

这是一个本地离线存档编辑工具，用于编辑游戏导出的 `.tyx` 存档文件。

## 使用流程

1. 在游戏中打开存档管理，导出需要编辑的存档。
2. 双击 `open-save-editor.bat` 打开编辑器。
3. 选择刚导出的 `.tyx` 文件。
4. 修改角色、时间、天气、背包等常用字段；复杂字段可在「高级 JSON 编辑」中修改。
5. 添加背包物品时，点击「加入/叠加物品」会打开物品选择窗口，可按分类筛选或按 ID/名称搜索。
6. 点击「导出加密 .tyx」。
7. 回到游戏，在存档管理里导入新的 `.tyx` 文件。

## 注意事项

- 编辑器不会直接修改游戏 localStorage，必须通过游戏内导入功能写回存档槽。
- 修改前建议保留原始 `.tyx` 备份。
- 物品选择窗口来自源码 `ITEMS` 目录；手动输入物品 ID 时仍需使用真实 ID，否则游戏加载时可能过滤掉无效物品。
- 高级 JSON 编辑可以改任意结构，改错会导致存档无法导入或加载异常。
